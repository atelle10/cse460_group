# SunDevil Connect by Michael Krasnik & Andrew Tellez

Instructions to run

## Required to have 
- Python 3.13 or higher
- pip (Python package manager)

## Setup Instructions

### 1. Navigate to Project Directory

```bash
cd sundevil_connect
```
### 2. Run the script

#### For Windows:
```cmd
run.bat
```

#### For macOS/Linux:
```bash
./run.sh
```
